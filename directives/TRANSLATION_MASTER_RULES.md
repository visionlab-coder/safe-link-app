# 건설 현장 번역 및 발음 규칙 (핵심 지침)

이 지침은 서비스 전체의 번역 품질과 일관성을 유지하기 위한 최우선 규칙입니다. 모든 번역 API는 아래 규칙을 엄격히 준수해야 합니다.

## 1. 현장 은어 맵핑 (Construction Jargon)
한국어 원문에 아래 단어가 포함된 경우, **번역문**에서는 표준어(안전 용어)로 변환하여 번역하되, **원문**의 뉘앙스를 이해하여 문맥에 맞게 처리합니다.

| 은어 | 표준어 변환 (번역 시 사용) | 의미 (참고) |
| :--- | :--- | :--- |
| **야리끼리** | 작업 할당제 | 할 일 마치면 퇴근하는 방식 |
| **공구리** | 콘크리트 | Concrete 타설 작업 |
| **데마찌** | 작업 대기 / 중단 | Standby (기상 악화 등으로 인한 대기) |
| **시마이** | 작업 종료 | Finish / 마감 |
| **곰방** | 운반 / 소운반 | 자재를 직접 들고 옮기는 작업 |
| **반생** | 구운 철사 | Annealed wire (결속용 철사) |
| **나라시** | 평탄화 작업 | Leveling (바닥 고르기) |
| **폼** | 거푸집 | Formwork |
| **루베** | 세제곱미터 | m³ (부피 단위) |
| **헤베** | 제곱미터 | m² (면적 단위) |
| **가타** | 어깨 | Shoulder / 옆면 |
| **하바키** | 발끝막이판 | Toe board |

## 2. 번역 원칙 (Translation Rules)
1. **비속어 제거:** 원문에 욕설이나 과격한 표현이 있어도, 번역문은 반드시 **정중하고 격식 있는 현장 안전 용어**로 출력합니다.
2. **안전 제일:** 안전 지시 사항(안전모, 추락 주의 등)은 누락 없이 명확하게 번역합니다.
3. **JSON 출력:** 모든 응용 프로그램 인터페이스(API)는 반드시 `{ "translation": "...", "pronunciation": "..." }` 형식을 유지해야 합니다.

## 3. 발음 표기 규칙 (Hangul Pronunciation) - 중요!
모든 외국어 번역 결과에 대해 **한국어(한글)로 차음**한 발음을 제공해야 합니다.
- **대상 언어:** 중국어, 베트남어, 태국어, 영어 등 한국어를 제외한 모든 언어.
- **표기 방식:** 한글로만 표기.
- **예시 (중국어):** "你好" -> "니 하오"
- **예시 (베트남어를 한국어로 번역 시):** 한국어 번역 결과에는 발음 표기를 비웁니다.
- **예시 (영어를 한국어로 번역 시):** 한국어 번역 결과에는 발음 표기를 비웁니다.
- **예시 (한국어를 영어로 번역 시):** "Wear helmet" -> "웨어 헬멧"
